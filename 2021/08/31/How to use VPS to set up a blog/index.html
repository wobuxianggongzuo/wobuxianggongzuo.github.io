心得

租用VPS (Virtual Private Server)

  • 租用步驟很簡單，填填資料便可以申請好一個帳號
  • 爬了一些文，主要還是看各家 VPS 的官網，網路上文章幾年前和現在的差異就有點大，但還是可以參考，篩選到最後剩下 linode 和 digitalocean
  • 界面色調我比較喜歡 digitalocean，但是官網對於 docker 和 wordpress 的資訊 linode 較多也較新，所以最後選擇了 linode
  • 日本伺服器延遲最低，毫不猶豫選下去！（疫情關係無法去日本玩也要將伺服器放在那🤣）
  • IP不能選擇，只能一直開新機器、刪掉這樣循環下去直到找到合意的 IP，每次新增一台機器需要 $0.01
  • 系統本來要選 Debian，但因為 Debian 的名稱是由開發者當時的女友 Debra 和 Ian Murdock 自己的名字合併而成的，現在女友已經是前妻了，所以改用比較熟悉的 Ubuntu

架設Wordpress

  • 會用到的服務主要有wordpress和 database
  • database 選擇 mariadb，因為 m1 也適用（隨時也可以換成 mysql）
  • 伺服器方面本來想用較新的nginx，但看到apache對於 PHP 支援較簡單，也不用多做其他設定，最後選擇了 apache
  • 因為想方便搬移、備份、在其他機器上重現、一鍵安裝等等，所以使用 dockerize 的方式來架設，但 docker 的資料會隨著服務的關閉而消失，所以需將資料掛載出來存在可以備份的地方
  • 資料掛載出來，以服務裡的 volumes 設定路徑在本機資料夾最為方便（若出現 DB 連線問題可稍微等一下即可）
  • 設定存放 volumes 的本機資料夾可以不存在，docker compose 會建立一個新的
