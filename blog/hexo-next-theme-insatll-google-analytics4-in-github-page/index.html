Posted on 2024-06-08 In blog
how to apply GA4 with github action

前言

這篇在紀錄如何將 Google Analytics 4（GA4），應用到 hexo 的 NexT 主題中
是給已經申請好 GA4 的帳號、建立好資源，但不知道如何讓 GA 開始運作的人看的
Hexo version：7.2.0
NexT version：8.20.0

太長，不要讀

  • 申請 Google Analytics 4 追蹤 ID（G-XXXXXXXXXX)
  • 追蹤程式碼，另存新檔案
  • 複製語法加入 github action

GA4 開始，追蹤碼的格式改為 G-XXXXXXXXXX
