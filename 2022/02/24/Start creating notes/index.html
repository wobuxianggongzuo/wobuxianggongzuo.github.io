Posted on 2022-02-24 Edited on 2023-12-15 In blog
how to build a auto deploy online notes from hugo and github action

notes

  • Hugo 建立線上筆記
  • Github action 自動發佈

Summary

步驟

  1. 開兩個 repo，一個存放筆記，一個放網站資料
  2. 安裝 Hugo
  3. 安裝 Hugo 主題
  4. 使用 ssh-keygen，將私鑰上傳到筆記 repo 的 secret，公鑰上傳到放網站資料 repo的 deploy key
  5. 建立 workflow

雜記

  • 隨時可以將整個資料夾刪除重來
  • workflow
    • 可以使用 deploy_key（不用像大多文章教學一樣使用personal_token）
    • 將./public push上另一個repo
  • 多重帳號
    • 使用 github desktop 可以登入與本機 global 不同的帳號 push
    • 使用終端機以及 vscode 建立新 markdown 文章
  • 使用 Hugo 的原因
    • Hexo 較慢，且有套件相依、更新等問題（但 Next 主題真的讚），較不好維護、轉移
    • Hugo 有 Next 主題，調整完有點複雜的設定之後，好用！！！
