Evergreen Note

  • Hugo 建立線上筆記
  • Github action 自動發佈

Summary

步驟

  1. 開兩個repo，一個存放筆記，一個放網站資料
  2. 安裝Hugo
  3. 安裝Hugo主題
  4. 使用ssh-keygen，將私鑰上傳到筆記repo的secret，公鑰上傳到放網站資料repo的deploy key
  5. 建立workflow

雜記

  • 隨時可以將整個資料夾刪除重來
  • workflow
    • 可以使用deploy_key（不用像大多文章教學一樣使用personal_token）
    • 將./public push上另一個repo
  • 多重帳號
    • 使用github desktop可以登入與本機global不同的帳號push
    • 使用終端機以及vscode建立新markdown文章
  • 使用Hugo的原因
    • Hexo較慢，且有套件相依、更新等問題（但Next主題真的讚），較不好維護、轉移
    • Hugo有Next主題，但設定繁複，沒有dark mode，忍痛捨棄QQ
